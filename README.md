# Fly print

这是一个边缘打印机管理软件，有如下功能和特点：
1）Windows和Linux环境中都可以运行
2）能够识别和连接同一个网络内的打印机
3）可以查询打印机的打印队列、状态等
4）可以给指定的打印机发起打印任务
5）将打印机相关的各种信息上传到服务端
6）接收服务端的打印任务，和文件

## 图片缩放模式说明
- `scale_mode=fit`：等比缩放并完整保留内容，不发生裁切。
- `scale_mode=actual`：仅在图片超出可打印区域时缩小，不主动放大。
- `scale_mode=fill`：等比缩放并尽量铺满可打印区域，允许在一侧发生裁切（常见于图片宽高比与纸张不一致时）。

## `scale_mode=fill` 最小验证步骤
1）准备一张宽高比明显异于目标纸张的图片（例如全景长图），在终端选择固定纸张后预览。  
2）将 `scale_mode` 设为 `fill`，确认预览区域已铺满且边缘存在可见裁切。  
3）发起打印并与预览对照，确认打印结果同样为铺满并保持同方向裁切。  
4）切回 `fit` 再次预览/打印，确认不再裁切，用于对照验证 `fill` 行为生效。
